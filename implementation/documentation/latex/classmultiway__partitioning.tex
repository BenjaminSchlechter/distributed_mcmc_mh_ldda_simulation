class to solve the multiway number partitioning problem 

Multi-\/way partitioning using the largest differencing method. This is an efficient approximation with statistically better results than a greedy algorithm. Implementation based upon\+: \href{https://en.wikipedia.org/wiki/Largest_differencing_method}{\texttt{ https\+://en.\+wikipedia.\+org/wiki/\+Largest\+\_\+differencing\+\_\+method}} \href{https://en.wikipedia.org/wiki/Multiway_number_partitioning}{\texttt{ https\+://en.\+wikipedia.\+org/wiki/\+Multiway\+\_\+number\+\_\+partitioning}}
